Gazebo Solidworks installer
Work on this project was stopped in ~2015 but the add-in still works in Solidworks (as of 2022)

Install the add-in by running the GazeboExporterInstaller.msi in the ../bin directory or downloading it first from :
https://usfirst.collab.net/sf/frs/do/viewRelease/projects.solidworks_robot_exporter/frs.solidworks_gazebo_export_plugin.solidworks_exporter_beta_1_0
